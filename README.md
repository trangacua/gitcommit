#  Alarm Clock Web App

Một ứng dụng website báo thức trực tuyến giao diện hiện đại, dễ sử dụng, giúp bạn quản lý thời gian và luôn thức dậy đúng giờ.

##  Tính năng nổi bật

* **Xem giờ thực tế:** Đồng hồ số hiển thị chính xác theo thời gian thực (hỗ trợ định dạng 12h/24h).
* **Cài đặt nhiều báo thức:** Cho phép thiết lập, bật/tắt hoặc xóa nhiều mốc giờ báo thức cùng lúc.
* **Âm thanh đa dạng:** Tùy chọn nhiều giai điệu báo thức khác nhau, từ nhẹ nhàng đến sôi động.
* **Chế độ báo lại (Snooze):** Cho phép tạm hoãn báo thức thêm 5-10 phút để bạn không bị lỡ giấc.
* **Giao diện Responsive:** Hiển thị mượt mà trên cả máy tính (Desktop), máy tính bảng (Tablet) và điện thoại (Mobile).
* **Chế độ Sáng/Tối (Light/Dark Mode):** Tự động hoặc thủ công chuyển đổi giao diện để bảo vệ mắt vào ban đêm.

##  Công nghệ sử dụng

Dự án này được xây dựng bằng các công nghệ:

* **Frontend:** HTML5, CSS3, JavaScript (hoặc React / Vue / Next.js nếu có)
* **Styling:** Tailwind CSS / Bootstrap (Xóa nếu không dùng)
* **Âm thanh:** Web Audio API
